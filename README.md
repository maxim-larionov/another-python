# another python

You can use python with another language. It's simple with my repository!

## to install

### run

<pre>
git clone https://github.com/maxim-larionov/another-python
</pre>

## to add langage

#### just write your language in JSON format (see example: another-python/russian.json)

## to use

### run

<pre>
python3 (path to repository/IDE.py) (path to script) (path to language in JSON format)
</pre>
